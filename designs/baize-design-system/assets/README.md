# Baize Tech Assets

当前已接入公司 Logo 原始文件，并派生出两个透明背景资产：

- `baize-logo.png`：用户提供的原始完整 Logo 备份。
- `baize-logo-lockup.png`：透明背景完整锁定图，用于品牌卡片、对外物料和下载包；不直接用于网页首屏。
- `baize-mark.png`：透明背景白泽图形 Mark，用于首页 Hero、Header、小尺寸图标和 favicon 候选。

后续仍建议补齐：

- 横版 Logo：白泽明理 Baize Tech。
- 方形 Logo / favicon。
- 文章或服务页可复用的低对比度路径纹理。
- 社交分享图模板。

小尺寸场景不要直接使用完整锁定图，否则中文和英文标语会不可读；优先使用 `baize-mark.png` 搭配文字字标。
