Use `Button` for primary consultation CTAs, secondary service exploration links, and quiet navigation actions.

```jsx
<Button variant="primary" size="lg">预约 AI 落地诊断</Button>
<Button variant="secondary" as="a" href="/services">查看服务</Button>
<Button variant="ghost">阅读明理笔记</Button>
```

Prefer concrete action copy. Avoid vague labels such as “了解更多” when the next step is known.
